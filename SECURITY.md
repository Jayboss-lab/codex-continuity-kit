# Security Policy

## Important: No Executable Code

This repository contains **documentation and markdown templates only**. It does not include executable code, build scripts, or dependencies. There are no API keys, credentials, or network connections within this repo.

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest on `main` | Documentation supported |
| Older commits | Not security-supported (this is a template repo) |

## Reporting a vulnerability

If you discover that a template or example accidentally contains:

- Private or proprietary information
- Hardcoded credentials or API keys
- Personal data
- Trademarked or confidential project details

Please open a **private security advisory** via GitHub or email the maintainers directly rather than opening a public issue.

We will remove the content promptly and issue a corrected release.

## Scope

Security reports should be limited to **accidental data leakage in templates or examples**. This is not an application or service, so traditional vulnerability categories (RCE, injection, etc.) do not apply.
